# VGT Civilian System Checker

**Von VisionGaia Technology | Open Source Edition**

Ein leichtgewichtiger Windows-Sicherheitsscanner der die grundlegendsten 
Schutzmaßnahmen prüft, die jedes System mindestens erfüllen sollte.

## Was wird geprüft?

- Windows Defender & Echtzeitschutz
- Firewall Status
- BitLocker Verschlüsselung
- Windows Update Konfiguration
- Grundlegende Systemhärtung

## Wichtiger Hinweis

Dies ist unsere **öffentliche Basisversion** – konzipiert für Endnutzer 
und kleine Organisationen die ihren Grundschutz prüfen möchten.

Unser vollständiger **VGT SafetySys™ Audit Engine** (38 Sicherheitsvektoren, 
9 Angriffskategorien, MITRE ATT&CK Mapping, Zertifikatsexport) 
ist **nicht öffentlich** und ausschließlich über 
[VisionGaia Technology](https://visiongaiatechnology.de) verfügbar.
